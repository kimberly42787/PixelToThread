# PixelToThread

PixeltoThread is a Python based project that transfroms digital images into cross-stitch patterns.

The goal of this project is to create an interactive web-based app where an image is uploaded and will return a cross-stitch pattern to follow with color keys based on the dimension and stitch density that the user selects.

Project Roadmap:
- [x] Get DMC threads data sets with corresponding RGB colors
- [x] Process image using K-means and get RGB colors
- [x] Map the RGB colors to DMC threads. Return image with the DMC colors
- [ ] Generate pattern with the DMC thread color keys
- [ ] Create an interactive web-based app

Optimization:
- [ ] Test out different image processing other than K-means
